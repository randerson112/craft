# craft clean

Removes the build directory and all compiled artifacts.

## Usage

```bash
craft clean
```

## Description

Deletes the `build/` directory generated by `craft build`, removing all compiled objects, executables, and CMake cache files. Reports how many files were removed and how many bytes were freed.

Craft walks up the directory tree to find the project or workspace root so you can run `craft clean` from any subdirectory.

### What gets deleted

- `build/` — all compiled artifacts, CMake cache, and object files

### What does NOT get deleted

- `.craft/deps/` — cloned git dependencies and their build artifacts are preserved
- `craft.toml` — never touched
- `CMakeLists.txt` — never touched
- Your source files — never touched

Because dependency builds are preserved in `.craft/deps/`, running `craft build` after `craft clean` only recompiles your project, not its dependencies. This makes clean builds significantly faster when you have large dependencies like SFML or raylib.

## Examples

```bash
craft clean
```

Output:
```
Removed 142 files, 24.3 MiB total.
```

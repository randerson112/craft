# craft workspace

Initializes a Craft workspace in the current or specified directory.

## Usage

```bash
craft workspace [path]
```

## Arguments

| Argument | Required | Description |
|----------|----------|-------------|
| `[path]` | no | Path to initialize the workspace. Defaults to current directory. |

## Description

Sets up a Craft workspace by scanning the directory for existing Craft projects, generating a `craft.toml` with a `[workspace]` section listing all detected members, and generating a root `CMakeLists.txt` that builds all members together.

### What it does

1. Scans the directory recursively for Craft projects
2. Generates `craft.toml` with detected members listed
3. Generates root `CMakeLists.txt`
4. Fetches all member git dependencies into the workspace `.craft/deps/`

### What it does NOT do

- Modify any member's existing `craft.toml`
- Modify any member's source files
- Overwrite an existing workspace `craft.toml`

## Examples

```bash
# Initialize workspace in current directory
craft workspace

# Initialize workspace at a specific path
craft workspace physics_engine
```

## Output

```
Detecting members...
Fetching dependencies...

Initialized Craft workspace at '/path/to/workspace'

Run 'craft build' to build the workspace
```

## After Initialization

The workspace root will contain:

```
physics_engine/
├── craft.toml          # generated workspace config
├── CMakeLists.txt      # generated by Craft
└── .craft/
    └── deps/           # shared dependencies
```

Review the generated `craft.toml` and verify all members were detected correctly. If any members are missing add them manually:

```toml
[workspace]
members = [
    "engine",
    "demos/demo1",
    "demos/demo2"
]
```

Then run `craft build` to build the workspace.

## Notes

- Members are detected by scanning for directories containing a `craft.toml` with a `[project]` section
- The workspace `craft.toml` contains only a `[workspace]` section, no `[project]` or `[build]`
- Run `craft build` from the workspace root or any subdirectory to build all members

# Commands Reference

Complete documentation for every Craft command.

## Project Management

| Command | Description |
|---------|-------------|
| [craft project](project.md) | Create a new project at a given path |
| [craft init](init.md) | Initialize a project in an existing directory |
| [craft workspace](workspace.md) | Initialize a workspace in an existing directory |
| [craft build](build.md) | Build a project or workspace |
| [craft run](run.md) | Run a compiled executable |
| [craft gen](gen.md) | Generate a file with boilerplate |
| [craft clean](clean.md) | Remove build artifacts |

## Dependency Management

| Command | Description |
|---------|-------------|
| [craft add](add.md) | Add a dependency |
| [craft remove](remove.md) | Remove a dependency |
| [craft update](update.md) | Update git dependencies |

## Configuration & Templates

| Command | Description |
|---------|-------------|
| [craft template](template.md) | Manage project templates |
| [craft config](config.md) | Manage global configuration |

## Craft Itself

| Command | Description |
|---------|-------------|
| [craft upgrade](upgrade.md) | Update Craft to the latest version |
| [craft help](help.md) | Show help for commands |
